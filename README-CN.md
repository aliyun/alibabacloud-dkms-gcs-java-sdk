## 依赖

- Java 1.8 or later
- Maven

## 安装

```
<dependency>
    <groupId>com.aliyun</groupId>
    <artifactId>alibabacloud-dkms-gcs-sdk</artifactId>
    <version>0.2.0</version>
</dependency>
```

## Build

```
mvn clean install -DskipTests
```
